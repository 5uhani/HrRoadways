## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/NishantRana07/HrRoadways.git
2. Navigate to the project directory and install dependencies:
  ```
    npm install
  ```
3. Run the development server to access the site locally: 
  ```
    npm run dev
  ```
